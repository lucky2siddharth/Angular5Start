1) Tutorial link:
    https://coursetro.com/courses/19/Learn-Angular-5-from-Scratch---Angular-5-Tutorial/lessons/3
2) Generate Project
    ng new <project_name> --style=scss --routing
3) Create a component
    ng g c <component_name>
4) To use ng-model we have to import FormsModule in app.module.ts
5) Generate Service
    ng g s <service_name>
6) Best way to share data between components by using 'behaviorSubject' of rxJS.
7) Build
    ng build ----> for normal build
    ng build --prod ------> for production build (highly optimised)
8) To run in production build
    ng build --prod --base-href="myurl"
9) Other way to deploy through GIT
    npm install -g angular-cli-ghpages
    Deploy then enter 'angular-cli-ghpage'
